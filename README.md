# VistoriaFácil - Três Lagoas

Sistema web desenvolvido para controle de alvarás e vistorias do Corpo de Bombeiros em Três Lagoas/MS.

## Descrição

O projeto tem como objetivo organizar o cadastro de estabelecimentos, o acompanhamento de vistorias e a verificação da situação dos alvarás, permitindo consultas internas por CNPJ.

## Objetivo

Desenvolver uma interface web para auxiliar no controle de estabelecimentos, status de alvará, datas de vistoria e validade dos registros.

## Funcionalidades

- Dashboard com visão geral.
- Listagem de estabelecimentos.
- Cadastro de novo estabelecimento.
- Consulta interna por CNPJ.
- Controle de status do alvará.
- Exibição da data da última vistoria.
- Exibição da validade do alvará.
