# Controle de Alvarás - Três Lagoas

Protótipo de sistema para gerenciamento de alvarás e vistorias do Corpo de Bombeiros, com painel administrativo e consulta pública simplificada por CNPJ.

## Objetivo

Desenvolver uma solução web para auxiliar no controle de alvarás, acompanhamento de vencimentos e consulta da regularidade básica de estabelecimentos.

## Funcionalidades previstas

- Cadastro de estabelecimentos
- Controle de alvarás e vistorias
- Consulta por CNPJ
- Visualização de status do alvará
- Alertas de vencimento
- Painel administrativo

## Tecnologias

- HTML
- CSS
- JavaScript
- GitHub
