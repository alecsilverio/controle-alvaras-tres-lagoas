# VistoriaFácil - Três Lagoas

Sistema web desenvolvido para apoiar o controle de alvarás e vistorias de estabelecimentos vinculados ao Corpo de Bombeiros em Três Lagoas/MS.

## Sobre o projeto

O VistoriaFácil foi criado com o objetivo de organizar, em uma interface web simples e funcional, o cadastro de estabelecimentos, o acompanhamento de vistorias e a verificação da situação dos alvarás. A proposta é centralizar informações importantes para consulta interna, permitindo uma visualização mais clara do status de regularidade de cada registro.

## Objetivo

Desenvolver uma aplicação web para auxiliar no controle de estabelecimentos cadastrados, datas de vistoria, validade de alvarás e situação geral dos registros, facilitando o acompanhamento e a consulta por CNPJ.

## Funcionalidades

- Dashboard com visão geral dos registros cadastrados.
- Indicadores de estabelecimentos em dia, a vencer e vencidos.
- Listagem completa dos estabelecimentos.
- Cadastro de novo estabelecimento.
- Consulta interna por CNPJ.
- Exibição da data da última vistoria.
- Exibição da validade do alvará.
- Controle visual do status do alvará.
- Interface responsiva e organizada para navegação simples.

## Como executar o projeto

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` em um navegador.
4. Navegue entre as páginas para acessar dashboard, cadastro, listagem e consulta.
